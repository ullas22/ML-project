
## 1. Create Python Virtual Environment
```
python -m venv .
```
or
```
virtualenv .
```

## 2. Activate Virtual Enviromment
Linux
```
source bin/activate
```

Windows
```
source Scripts/activate
```

## 3. Install Dependencies 
```
pip install -r requirements.txt
```

## 4. Run the app  
```
flask run
```
Check http://localhost:5000/

## 5. Exit the virtual environment
```
deactivate
```
